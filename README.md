
# SIM - Mantenimiento Integral (MVP)

Este es el sitio web de SIM, diseñado para mantenimiento industrial, comercial y residencial en Guadalajara.

## Cómo desplegar en Netlify (Gratis)

1. Descarga todos los archivos de este proyecto.
2. Ve a [Netlify Drop](https://app.netlify.com/drop).
3. Arrastra la carpeta del proyecto directamente al navegador.
4. Una vez subido, ve a **Site Settings > Domain Management** para elegir un nombre de subdominio gratuito (ej. `sim-gdl.netlify.app`).

## Tecnologías utilizadas
- React 19
- Tailwind CSS (Diseño responsivo y Modo Oscuro)
- Vite (Compilador ultra rápido)
- Formspree (Gestión de formularios de contacto sin backend)
